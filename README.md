# MindForge AI

MindForge AI is an AI-powered personalized learning platform designed to help users learn smarter through roadmaps, quizzes, recall systems, and concept analysis.

## Features

- Personalized learning roadmaps
- AI-generated quizzes
- Quiz evaluation system
- Knowledge Decay Tracker
- Recall Cards
- Progress Dashboard
- Visual Concept Cards
- Concept Confusion Detector

## Tech Stack

### Backend

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Qdrant
- Groq / LLaMA 3

### Frontend

- React.js
- JavaScript
- CSS
